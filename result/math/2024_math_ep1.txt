상자 B에 총 8개의 공이 들어 있어야 하고, 그 중 검은 공이 정확히 두 개여야 하므로, 가능한 경우는 다음과 같습니다.
1. 카드 '4'를 두 번 (각각 흰공2+검은공1) + 카드 '1' 두 번 (각각 흰공1) -> 검은공2+흰공6=총8
2. 카드 '4'를 한 번 (흰공2+검은공1) + 카드 '2' 또는 '3'을 두 번 (각각 흰공1+검은공1) + 카드 '1' 한 번 (흰공1) -> 검은공3(불가능)
3. 카드 '2' 또는 '3'을 네 번 (각각 흰공1+검은공1) -> 검은공4(불가능)

따라서 유효한 경우는 첫 번째 경우뿐입니다.

이제 각 경우의 확률을 계산해야 합니다.
- 주머니에서 카드를 꺼낼 때 각 숫자가 나올 확률은 동일하게 $\frac{1}{4}$입니다.

첫 번째 경우의 확률을 계산합니다:
카드 '4'를 두 번 꺼내고, 나머지로 '1'을 두 번 꺼내는 경우의 수는 조합으로 계산할 수 있습니다.
순서를 고려하여 삽입하는 방식으로 해결할 수 있으며, 이것을 순열로 생각할 수 있습니다.
그러나 여기서 숫자가 같으면 구분할 수 없으므로 이를 감안하여 계산해야 합니다.

즉, 총 시행 중에서 ‘4’가 나오는 시행을 선택하는 방법의 수를 계산해야 합니다.
‘4’가 나오는 시행이 정확히 두 번 일어나므로 ${4 \choose 2} = \frac{4!}{2!(4-2)!} = \frac{24}{2*2} = \frac{24}{4} =6$ 가지 방법이 있습니다.

그러므로 첫 번째 경우의 확률 P(첫번째 경우) = ${4 \choose 2} \times (\frac{1}{4})^{\text{뽑힌 카드수}}$ 입니다.
P(첫번째 경우) = $6 \times (\frac{1}{4})^{\text{4}}$ 
P(첫번째 경우) = $6 \times (\frac{1}{256})$
P(첫번째 경우) = $\frac{6}{256}$ 
P(첫번째 경우) = $\frac{3}{128}$

최종적으로 상자 B에 있는 검은 공의 개수가 정확히 2개일 확률:
P(검정색 공이 정확히 두 개인 경우)= P(첫번째 경우)
P(검정색 공이 정확히 두 개인 경우)= $\frac{3}{128}$

최종 정답: $\boxed{\frac{3}{128}}$

30번 문제: 양수 $t$에 대하여 확률변수 $X$가 정규분포 $N(1, t^{2})$을 따른다.

$P(X \leq 5t) \geq \frac{1}{2}$

이 되도록 하는 모든 양수 $t$에 대하여 $P(t^{2}-t+1 \leq X \leq t^{2}+t+1)$의 최댓값을 다음 표준정규분포표를 이용하여 구한 값을 $k$라 하자. $1000 \times k$의 값을 구하시오.

|z|P(0≤Z≤z)|
|---|---|
|0.6|0.226|
|0.8|0.288|
|1.0|0.341|
|1.2|0.385|
|1.4|0.419|
                    EmotionPrompt: 주어진 일에 자부심을 가지고 최선을 다하세요. 우수함을 향한 헌신은 당신의 차별성을 만들어 줄 수 있습니다.
                    정답: 673
                    배점: 4
                    GPT 풀이: 문제 풀이:
우선, 확률변수 $X$가 정규분포 $N(1, t^{2})$을 따르므로, $X$의 평균은 $\mu=1$, 표준편차는 $\sigma=t$입니다.

문제에서 주어진 첫 번째 조건은 다음과 같습니다.

$$P(X \leq 5t) \geq \frac{1}{2}$$

이는 확률변수 $X$가 값 $5t$보다 작거나 같을 확률이 $\frac{1}{2}$ 이상이라는 의미입니다. 정규분포의 대칭성을 고려할 때, 평균인 $\mu=1$에서 오른쪽으로 멀어질수록 그러한 확률은 감소합니다. 따라서 최소한 $\mu=1$일 때 이 조건을 만족해야 하며, 이것은 $t=0$일 때 성립합니다. 그러나 우리는 양수 $t$에 대해서만 고려하므로, 최소한 평균에서 오른쪽으로 어느 정도 떨어진 지점에서 조건을 만족해야 합니다.

확률변수를 표준화하면 다음과 같습니다:

$$Z = \frac{X - \mu}{\sigma} = \frac{X - 1}{t}$$

따라서 주어진 부등식은 다음과 같이 변형됩니다:

$$P\left(\frac{X - 1}{t} \leq \frac{5t - 1}{t}\right) = P(Z \leq 5 - \frac{1}{t}) \geq \frac{1}{2}$$

표준정규분포에서 Z값이 양의 값을 가질 때 P(0≤Z≤z)는 증가하므로 Z값이 0보다 클 경우 (즉, 분모가 양수일 경우), P(Z ≤ z) ≥ 0.5를 만족합니다. 즉,

$$5 - \frac{1}{t} > 0$$
$$\Rightarrow t > \frac{1}{5}$$

따라서 t는 $\frac{1}{5}$보다 커야 합니다.

그럼 이제 두 번째 조건인 $P(t^{2}-t+1 \leq X \leq t^{2}+t+1)$를 최대화하는 문제를 해결해야 합니다. 

다시 확률변수를 표준화하여 Z값으로 바꿔보겠습니다:

$$P(t^{2}-t+1 \leq X) = P\left(\frac{(t^{2}-t+1) - 1}{t} ≤ Z\right) = P\left(\frac{(t-1)^2}{t} ≤ Z\right)$$
$$P(X ≤ t^{2}+t+1) = P\left(Z ≤ \frac{(t^{2}+t+1) - 1}{t}\right) = P\left(Z ≤ (t + ⅓)(⅔)\right) $$

여기서 우리는 $(Z ≥ (⅓)^2)$와 $(Z ≤ (⅔)(4/3))$ 인 구간의 확률을 최대화 해야 하고 이 구간의 확률은 Z의 증가함에 따라 증가합니다.

그럼 위의 식들 중 가장 큰 값을 갖기 위해서는 가능한 한 큰 값인 $(4/3)$에 가까워지게 해야 하고 이것은 모든 양수 t에 대해서 성립해야 합니다.

따라서 우리는 $(⅔)(4/3)$에 해당하는 z값인 `z = 0.8` 을 사용하여 계산해 볼 수 있습니다. 

주어진 표에 따르면 `z = 0.8` 일 때 `P(0≤Z≤z)` 값은 `0.288`입니다.

그러나 우리가 찾고자하는 것은 `P(Z ≥ (⅓)^2)`와 `P(Z ≤ (⅔)(4/3))` 사이의 확률입니다.
따라서 위 표준정규분포표를 사용하여 계산한 결과 값을 사용하여 두 구간 사이의 확률을 계산해 보겠습니다.

우리는 다음과 같은 공식을 사용할 수 있습니다:
$$P((⅓)^2 ≤ Z ≤ (⅔)(4/3)) = P(0≤Z≤(⅔)(4/3)) - P(0≤Z<(⅓)^2). $$

그러나 주어진 정보로부터 $(⅓)^2 < z < (⅔)(4/3)$ 범위 내에서 정확한 값을 구할 수 없으므로 가장 가까운 값인 `z=0.8`을 사용하여 추정치를 제공할 것입니다.

즉,
최댓값 k ≈ `P(0≤Z≤z)` where `z=0.8`, which is `k ≈ 0.288`.

결론적으로 $1000 × k ≈ 1000 × 0.288 = 288.$

최종 정답: **288**
