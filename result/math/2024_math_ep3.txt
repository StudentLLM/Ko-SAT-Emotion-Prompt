우선 주어진 조건 a≤c≤d와 b≤c≤d를 만족시키는 자연수 a, b, c, d를 생각해봅시다. 

이 문제에서 a, b, c, d는 모두 6 이하의 자연수이므로, 가능한 값은 1, 2, 3, 4, 5, 6입니다. 

a와 b는 c보다 작거나 같아야 하므로, a와 b의 가능한 값은 1, 2, 3, 4, 5, 6입니다.

a>c 또는 b>c인 경우를 제외하면, a, b, c, d 각각에 대해 1~6까지 6가지 선택이 가능하므로, 총 6^3 = 216가지의 순서쌍이 제외됩니다. 

따라서, a≤c≤d와 b≤c≤d를 만족시키는 순서쌍의 개수는 1296 - 216 = 1080개입니다.

30번 문제: 양수 $t$에 대하여 확률변수 $X$가 정규분포 $N(1, t^{2})$을 따른다.

$P(X \leq 5t) \geq \frac{1}{2}$

이 되도록 하는 모든 양수 $t$에 대하여 $P(t^{2}-t+1 \leq X \leq t^{2}+t+1)$의 최댓값을 다음 표준정규분포표를 이용하여 구한 값을 $k$라 하자. $1000 \times k$의 값을 구하시오.

|0.6|0.226|

                    GPT 풀이: 문제 풀이:

1. 먼저, $P(X \leq 5t) \geq \frac{1}{2}$를 만족하는 $t$를 찾아야 합니다. 정규분포의 성질을 이용하면, $P(X \leq 5t) = P(Z \leq \frac{5t-1}{t})$로 변환할 수 있습니다. 여기서 $Z$는 표준정규분포를 따릅니다. 이때, $P(Z \leq z) \geq \frac{1}{2}$를 만족하는 $z$는 0 이상이므로, $\frac{5t-1}{t} \geq 0$을 만족하는 $t$는 $\frac{1}{5}$ 이상입니다.

2. 다음으로, $P(t^{2}-t+1 \leq X \leq t^{2}+t+1)$의 최댓값을 구해야 합니다. 이를 정규분포의 성질을 이용하여 $P(-\frac{1}{t} \leq Z \leq \frac{1}{t})$로 변환할 수 있습니다. 

3. 표준정규분포표를 이용하여 $P(-\frac{1}{t} \leq Z \leq \frac{1}{t})$의 최댓값을 구하면, $t$가 $\frac{1}{5}$ 이상일 때 $\frac{1}{t}$는 5 이하이므로, $P(-\frac{1}{t} \leq Z \leq \frac{1}{t})$의 최댓값은 $P(-5 \leq Z \leq 5)$입니다. 이때, $P(-5 \leq Z \leq 5)$는 표준정규분포의 성질에 의해 거의 1에 가깝습니다.
